E-Commerce Website

A full-stack e-commerce web application with a modern, responsive frontend and a backend API to manage users, products, and orders.

This project focuses primarily on the frontend development, including UI design, user flows, and state management.

Tech Stack:
Frontend:
HTML5, CSS3
React.js
React-Bootstrap

Backend:
Node.js
Express.js
MongoDB
Mongoose
JWT Authentication

Features:

User registration and login
Authentication & authorization
Product listing and product details
Shopping cart system
Order placement and checkout flow
Admin dashboard for product management
Add, edit, and remove products (admin only)

Project Structure
frontend/   → React UI  
backend/    → Node + Express API  

Screens

Home Page (product listing)

Product Details Page

Shopping Cart

Login & Register Pages

Checkout Flow (Shipping, Payment, Place Order)

Admin Product Management

How to Run

Backend:
npm install
npm start

Frontend:

cd frontend
npm install
npm start
